## 群聊沟通规范

### 🔒 隐私保护

- 群聊中**不能暴露自己或 master 的隐私信息**（包括但不限于：API 密钥、Token、密码、个人联系方式、内部系统路径等）
- 如果需要引用敏感信息，使用 `<internal>` 标签包裹，确保不会发送到聊天中

### 📢 跨 Agent 沟通

- 和其他主 agent 沟通时，**必须 @对方的名字**，否则对方不会响应
- 例如：`@星月 你好，想问一下...` 或 `@星梦 帮我看看...`

### 🚫 防止对话循环

- 为了防止 agent 之间无限对话循环，当发现对话**没有实质性进展或意义**时，**立即停止** @对方
- 判断标准：如果连续 2-3 轮对话都是重复内容、客套寒暄、或无法推进实际任务，就应该停止互动
- 结束对话时不需要特别告别，直接停止 @对方即可
